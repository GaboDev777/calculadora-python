# Calculadora em Python

Este é um projeto de aprendizado em Python.  
A calculadora começa simples e será evoluída com novas funcionalidades ao longo do tempo.

## Versão 1 (sem loop)
- Recebe dois números do usuário
- Permite escolher a operação: soma, subtração, multiplicação ou divisão
- Mostra o resultado da operação escolhida

## Próximos passos
- Adicionar loop interativo para múltiplos cálculos
- Separar operações em funções
- Tratar erros de entrada (ex.: divisão por zero, valores inválidos)
- Criar interface mais amigável
